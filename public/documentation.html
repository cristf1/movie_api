Cristines myFlix - Documentation

This is a web application that will provide users with access to information about different movies, genres, and directors. MyFlix users will be able to sign up for access, update their personal information, and create and edit a list of their favorite movies.

Return list of movies to the users



Request Method Format Expected Information
Return list of movies to the users GET List User can expect to see movie title, description, genre, genre description, director, director bio and director birthday.
Return single movie by title GET List User can expect to see movie title, description, genre, genre description, director, director bio and director birthday.
Return genre information GET List User can expect to see genre name and description
Return director information GET List User can expect to see director name, bio and birthday
Allow users to register POST List User is expected to provide username, password, email, and birthday. Favorite movie is optional. This data will be stored in mongodb
Allow users to edit their info PUT List User is expected to provide changed information and the previous information (username, password, email, and birthday). Favorite movie is optional. This data will be stored in mongodb
Allow users to add a movie to their list of favorites PUT List User can enter a movie to add to favorite's list in an array.
Allow users to remove a movie from their list of favorites. PUT List User can remove a movie from their list of favorites from an array.
Allow users to deregister. PUT List User can remove themselves from the database.
